# جسمانی مصنوعی ذہانت اور ہیومنوائڈ روبوٹکس کا تعارف

## نظریہ

جسمانی مصنوعی ذہانت (Physical AI) مصنوعی ذہانت اور جسمانی دنیا کا اتصال ہے، جو مشینوں کو اپنے ماحول کا ادراک، منطق اور تعامل کرنے پر مرکوز ہے۔ روایتی مصنوعی ذہانت کے برعکس جو بنیادی طور پر ڈیجیٹل دائرے میں کام کرتی ہے، جسمانی مصنوعی ذہانت حس، ترسیل اور عمل کو جامع طور پر ضم کرتا ہے تاکہ روبوٹ جسمانی دنیا میں سمجھدارانہ طریقے سے نیویگیشن، مینیپولیشن اور تعامل کر سکیں۔

### جسمانی مصنوعی ذہانت کیا ہے؟

جسمانی مصنوعی ذہانت درج ذیل اہم صلاحیات پر مشتمل ہے:
- **ادراک**: کیمرے، لیڈار، IMU، اور دیگر سینسرز کے ذریعے جسمانی ماحول کا جائزہ لینا اور سمجھنا
- **منطق**: جگہی رشتے، طبعیات اور وسائل کو سمجھنے کے لیے حسی ڈیٹا کی ترسیل
- **عمل**: اشیاء کو ہیرا پھیری اور ماحول کے ساتھ نیویگیٹ کرنے کے لیے جسمانی اعمال انجام دینا
- **جثہ**: ایک جسمانی شکل ہونا جو دنیا سے تعامل کر سکتی ہو
- **سیکھنا**: ماحول کے ساتھ تعامل کے ذریعے رویے میں بہتری لانا

یہ شعبہ ہیومنوائڈ روبوٹکس میں اہم ہے، جہاں مقصد ایسے روبوٹ تخلیق کرنا ہے جن میں انسان نما صلاحیات ہوں تاکہ وہ انسانی ماحول اور اوزار کے ساتھ تعامل کر سکیں۔

### ہیومنوائڈ روبوٹکس کا ابھرنا

ہیومنوائڈ روبوٹس اہمیت حاصل کر رہے ہیں کیونکہ:
- **انسان اور مرکزی ڈیزائن**: انسانی ماحول (عمارتیں، اوزار، گاڑیاں) انسانی ابعاد اور صلاحیات کے لیے ڈیزائن کیے گئے ہیں
- **سماجی قبولیت**: انسان نما شکل اور سلوک بہتر انسان-روبوٹ تعامل کے لیے آسانی فراہم کر سکتا ہے
- ** versatility **: انسان نما شکل وہ کام انجام دے سکتی ہے جو انسان کر سکتے ہیں
- **جنرل پurpose**: مخصوص کام کے بجائے جنرل پurpose روبوٹس کا امکان

### جسمانی مصنوعی ذہانت اور ہیومنوائڈ روبوٹکس میں اہم چیلنج

1. **ہیومنوائڈ کنٹرول کی پیچیدگی**: متعدد ڈگریوں کی آزادی کے ساتھ توازن، چلنا، اور ہیرا پھیری
2. **ریل ٹائم پروسیسنگ**: توازن قائم رکھنے اور متحرک ماحول کا جواب دینے کے لیے فیصلہ کرنا
3. **حفاظت**: یقینی بنانا کہ روبوٹ انسانوں کے ساتھ محفوظ طریقے سے تعامل کر سکیں
4. **قیمت**: ہیومنوائڈ روبوٹس کو معاشی طور پر جائز بنانا
5. **اخلاقیات اور رازداری**: معاشرے میں انسان نما ایجنٹس کے بارے میں خدشات کا سامنا کرنا

### جسمانی مصنوعی ذہانت اور ہیومنوائڈ روبوٹکس کے اطلاقیات

- **مددگار دیکھ بھال**: بزرگ افراد یا معذور افراد کے روزمرہ کے کاموں میں مدد کرنا
- **صنعتی کام**: پیداوار میں خطرناک یا دہرے کام انجام دینا
- **صحت کی دیکھ بھال**: طبی کارروائیوں یا مریضوں کی دیکھ بھال میں مدد کرنا
- **تعلیم**: تدریس کے معاون یا تعلیمی ساتھیوں کے طور پر خدمات فراہم کرنا
- **تفریح**: تعاملی تجربات فراہم کرنا
- **گھریلو خدمت**: گھر کے کاموں میں مدد کرنا

import numpy as np
from dataclasses import dataclass
from typing import Tuple

@dataclass
class RobotState:
    """ہیومنوائڈ روبوٹ کی حالت کی نمائندگی کرتا ہے"""
    position: Tuple[float, float, float]  # x، y، z کوآرڈینیٹس
    orientation: Tuple[float, float, float, float]  # کوارٹرینون (w, x, y, z)
    joint_angles: np.ndarray  # جوائنٹ اینگلوں کا اری
    joint_velocities: np.ndarray  # جوائنٹ کی رفتاروں کا اری
    linear_velocity: Tuple[float, float, float]  # ورلڈ فریم میں رفتار
    angular_velocity: Tuple[float, float, float]  # زاویہ کی رفتار

@dataclass
class HumanoidModel:
    """ہیومنوائڈ روبوٹ کنیمیٹکس کے لیے بنیادی ماڈل"""
    # جوائنٹ کے نام اور کنفیگریشن
    joint_names = [
        'left_hip', 'left_knee', 'left_ankle', 'right_hip', 'right_knee', 'right_ankle',
        'left_shoulder', 'left_elbow', 'left_wrist', 'right_shoulder', 'right_elbow', 'right_wrist',
        'neck', 'waist', 'torso'
    ]

    def __init__(self):
        self.num_joints = len(self.joint_names)
        self.state = RobotState(
            position=(0.0, 0.0, 0.8),  # کھڑے ہونے کی حالت
            orientation=(1.0, 0.0, 0.0, 0.0),  # غیر جانبدار سمت
            joint_angles=np.zeros(self.num_joints),
            joint_velocities=np.zeros(self.num_joints),
            linear_velocity=(0.0, 0.0, 0.0),
            angular_velocity=(0.0, 0.0, 0.0)
        )

    def move_to_pose(self, target_pose):
        """ہدف کے پوز کو حاصل کرنے کے لیے جوائنٹ اینگل کا حساب لگائیں"""
        # حقیقی نافذ کاری میں یہ معکوس کنیمیٹکس استعمال کرے گا
        print(f"پوز تک جانے کا حساب لگایا جا رہا ہے: {target_pose}")

        # حقیقت میں، اس میں زیادہ ترقی یافتہ طبعیات کی تبدیلی استعمال کی جائے گی

# مثال کا استعمال
if __name__ == "__main__":
    robot = HumanoidModel()

    # دائیں ہاتھ کے لیے ایک سادہ ہدف کی وضاحت کریں
    target_pose = (0.5, 0.0, 1.0)  # 0.5 میٹر سامنے، سینے کی اونچائی پر

    # درکار جوائنٹ اینگل کا حساب لگائیں
    joint_commands = robot.move_to_pose(target_pose)

## محاکات

اس باب کے لیے، ہم اپنی محاکات کے لیے گیزبو سیمولیٹر استعمال کریں گے۔ گیزبو حقیقی طبعیات کی محاکات فراہم کرتا ہے جو جسمانی مصنوعی ذہانت کی ترقی کے لیے ضروری ہے۔ یہاں دیکھیں کہ آپ اپنی پہلی محاکات کیسے ترتیب دیں:

### ماحول کی ترتیب

1. **گیزبو انسٹال کریں**: سرکاری گیزبو ویب سائٹ پر جائیں اور اپنے آپریٹنگ سسٹم کے لیے انسٹالیشن ہدایات کو فالو کریں
2. **ROS 2 ہمبول ہاکسبل انسٹال کریں**: یہ ہیومنوائڈ روبوٹکس کے لیے تجویز کردہ ROS 2 تقسیم ہے
3. **ورک سپیس کی ترتیب**: اپنے روبوٹکس پروجیکٹس کے لیے ایک کیٹکن ورک سپیس تشکیل دیں

### پہلی محاکات: بنیادی ہیومنوائڈ ماڈل

اپنی پہلی محاکات چلانے کے لیے:

```bash
# ٹرمنل 1: گیزبو شروع کریں
ros2 launch gazebo_ros empty_world.launch.py

# ٹرمنل 2: ہیومنوائڈ ماڈل کو اسپون کریں (مثال کے طور پر)
ros2 run gazebo_ros spawn_entity.py -entity my_robot -file path/to/robot/model.urdf -x 0 -y 0 -z 1
```

### محاکات کے مشقیں

1. **ملاحظہ کریں**: مشاہدہ کریں کہ روبوٹ محاکاتی ماحول میں کیسے اچار کرتا ہے
2. **تبدیل کریں**: ماحول کے پیرامیٹرز (گریویٹی، فرکشن، وغیرہ) کو تبدیل کریں اور اثرات دیکھیں
3. **مداخلت کریں**: ROS 2 ٹوپکس کا استعمال کر کے حکم دیں اور جوابات دیکھیں

## مشقیں

1. ** 개념ی سمجھ **:
   - روایتی مصنوعی ذہانت اور جسمانی مصنوعی ذہانت کے درمیان فرق کی وضاحت کریں
   - روبوٹس کے لیے ہیومنوائڈ شکل کے فوائد کیا ہیں؟
   - ہیومنوائڈ روبوٹس کے لیے تین حقیقی دنیا کے اطلاقیات کی وضاحت کریں

2. ** تکنیکی نافذ کاری **:
   - کم از کم 15 جوائنٹس کے ساتھ ہیومنوائڈ روبوٹ کی نمائندگی کے لیے ایک ڈیٹا سٹرکچر بنائیں
   - ایک فنکشن نافذ کریں جو جوائنٹ اینگلوں کی بنیاد پر اینڈ ایفکٹر (ہاتھ) کی 3D پوزیشن کا حساب لگاتا ہے
   - ایک سادہ اسٹیٹ مشین ڈیزائن کریں جو کسی بنیادی کام (جیسے ہاتھ ہلانا) کو انجام دینے کے لیے ہیومنوائڈ روبوٹ کے لیے ہو

3. ** تحقیق اور تجزیہ **:
   - ایک تجارتی طور پر دستیاب ہیومنوائڈ روبوٹ تلاش کریں اور اس کے ڈیزائن کے انتخاب کا تجزیہ کریں
   - جسمانی مصنوعی ذہانت پر ایک حالیہ پیپر (2023-2025) تلاش کریں اور اس کی اہم شراکت کا خلاصہ دیں
   - مختلف مینوفیکچرز والے ہیومنوائڈ روبوٹس کا موازنہ کریں اور مشترکہ ڈیزائن پیٹرنز کی شناخت کریں

4. ** محاکات چیلنج **:
   - ایک سادہ ہیومنوائڈ روبوٹ کا URDF ماڈل بنائیں (6+ جوائنٹس کے ساتھ)
   - گیزбо میں روبوٹ کو کسی شے تک پہنچنے کی محاکات کریں
   - استعمال کردہ کنیمیٹک چین اور جوائنٹ کنٹریکٹس کو دستاویز کریں

5. ** تنقیدی سوچ **:
   - ہیومنوائڈ روبوٹس کو انسانی ماحول کے لیے محفوظ بنانے میں کیا چیلنج ہیں؟
   - معاشرے میں ہیومنوائڈ روبوٹس کے اخلاقی اثرات پر تبادلہ خیال کریں
   - مختلف اطلاقیات کے لیے ہیومنوائڈ روبوٹس کی معاشی موزونیت کا جائزہ لیں
