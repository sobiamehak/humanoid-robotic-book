# 👋 Welcome to the Frontend

This section contains documentation for the frontend implementation of the Physical AI & Humanoid Robotics textbook application.

## 🚀 About This Frontend

The frontend is built with **Docusaurus** and provides:

- Interactive textbook content
- User authentication and profiles
- Personalized learning paths
- Urdu translation capabilities
- Embedded RAG chatbot for questions

## 🛠️ Key Components

- **Authentication System**: Using Auth.tsx and Profile.tsx for user management
- **Documentation Pages**: Docusaurus-based textbook content
- **Interactive Elements**: Embedded tools and features within textbook chapters
